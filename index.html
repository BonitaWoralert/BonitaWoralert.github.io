DirectX11 Graphics Project

Year 2 Semester 1

This project was completed as an introduction to DirectX11 and graphics programming. I implemented features such as lighting, model loading, and a moveable camera.

I used HLSL to create ambient, diffuse, and specular lighting. These are calculated per pixel using Phong shading. In the future I'd like to add more features to the camera, for example, multiple cameras that can be switched between.
