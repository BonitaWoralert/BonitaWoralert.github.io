Procedural Generation of Trees - Work in Progress

A supporting artefact to my dissertation for Games Programming.

Exploring the usage of Space Colonisation Algorithm for the real time procedural generation of trees and plants.

Created in Unity 2022.3.4f1 and Visual Studio 2022. Currently working on the geometry generation - cylinder branches of the same size are now generated, but the radius should later be scaled based on distance from the root node. This should result in the tree having a large trunk and small branches at the top, giving a more realistic appearance.
